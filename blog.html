Frequently Asked Questions

The `var` keyword has a function scope. but `let` and `const` have block scope.
`let` allows redeclaration in the same block, and `const` declares a constant that cannot be reassigned after initialization.

`null` is typically used to represent the intentional absence of any object value. `undefined` is a variable that has been declared but has not yet been assigned a value. It is also the default return value of functions.

REST API mean (Representational State Transfer) which is an architectural style for designing networked applications. It uses a stateless, client-server communication model, and data is transferred over standard HTTP methods
